# LPC Spritesheet Generator - Complete Hash Parameter Catalog

## URL Format

Base URL: `https://liberatedpixelcup.github.io/Universal-LPC-Spritesheet-Character-Generator/`

Hash format: `#sex=<bodyType>&<type_name>=<ItemName_variant>&...`

- `sex` (body type): `male`, `female`, `muscular`, `pregnant`, `teen`, `child`
- Item names have spaces replaced with underscores
- Variant is appended after an underscore: `ItemName_variant`
- Example: `#sex=male&body=Body_Color_light&head=Human_Male_light&weapon=Longsword`
