# Spell Icons — Midjourney Prompts

All ability icons for The Hollows combat system. Each icon should be generated as a 256x256 PNG.

## Prompt Template

```
Dark fantasy icon painting, square ability icon format, dark background with glowing [ELEMENT] accent, painterly brushwork, centered [SYMBOL/EFFECT] --ar 1:1 --s 200 --no text characters words letters
```
